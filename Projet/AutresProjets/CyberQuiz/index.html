Site CyberQuiz

Création d'un site de quiz sur la cybersécurité. Ce quiz a été créé dans le but de tester et d'améliorer les compétences de chaque employé de l'entreprise. De plus, ce test a été traduit en anglais pour les autres pays. Malheureusement je ne peux présenter aucune donnée ou photo de ce projet car il est confidentiel.

Fonctionnement du Site

Ce site a été codé en HTML, CSS et JavaScript. De plus, les questions ont été codées en JSON pour pouvoir les utiliser avec le JavaScript. Il y a en tout 50 questions qui peuvent être utilisées. Dans le test et pour chaque personne, 12 questions sont posées. Pour le choix des questions, lors du questionnaire, les questions sont tirées au sort. A la fin du questionnaire, la personne doit avoir 8/12 pour réussir le test et avoir un diplôme créé par nos soins. Sinon, la personne peut repasser les tests. De plus, l'employé doit s'authentifier avant que les résultats soient envoyés pour récupérer les informations.
