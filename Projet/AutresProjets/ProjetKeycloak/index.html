Projet Keycloak - Authentification SSO

Mise en place d'une infrastructure d'authentification unique (SSO) avec Keycloak dans un environnement virtualisé. Ce projet a permis de créer une solution d'authentification centralisée pour les différents services de l'entreprise. L'architecture repose sur trois machines virtuelles interconnectées et intègre une application web Symfony sécurisée par Keycloak.

Fonctionnement du Projet

Le système d'authentification Keycloak a été déployé dans une architecture à trois machines virtuelles : une VM Debian 12 hébergeant le serveur Keycloak et l'application Symfony, une VM Windows Server avec Active Directory pour la gestion des utilisateurs, et une VM Windows 11 pour tester l'accès client. L'application web développée en Symfony permet aux utilisateurs de se connecter via le service d'authentification Keycloak, qui vérifie les identifiants auprès de l'Active Directory. Cette architecture assure une authentification sécurisée et centralisée pour tous les services de l'entreprise.
