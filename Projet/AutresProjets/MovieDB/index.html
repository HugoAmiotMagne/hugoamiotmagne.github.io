Projet MovieDB - Application Mobile

Conception et développement d'une application Android de gestion de films et séries dans le cadre d'un projet de BTS SIO. Cette application permet aux utilisateurs de consulter des informations détaillées sur des films et séries en utilisant une API externe pour récupérer les données. Dans ce projet en binôme, j'étais responsable de la section films et de l'intégration des données provenant de l'API.

Description du Projet

L'application MovieDB offre une interface intuitive permettant aux utilisateurs de parcourir une large base de données de films et séries (20 films et 20 series). Chaque élément dispose d'une fiche détaillée comprenant le synopsis, la date de sortie, les acteurs principaux, la note moyenne et une affiche. L'application est conçue avec une architecture moderne permettant une navigation fluide entre les différentes sections et une mise à jour régulière des contenus grâce à l'intégration d'une API externe. La gestion des requêtes réseau et le traitement des données JSON sont optimisés pour garantir une expérience utilisateur réactive même avec une connexion internet limitée.

Réalisation de l'Application

Ce projet a été développé en collaboration avec un autre étudiant et a nécessité plusieurs étapes de conception et développement:

  • Analyse des besoins : définition des fonctionnalités principales et de l'expérience utilisateur cible
  • Maquettage : création des wireframes et design de l'interface utilisateur
  • Développement Android : programmation en Java avec Android Studio
  • Intégration de l'API : implémentation des requêtes HTTP pour récupérer les données des films
  • Traitement des données : parsing des réponses JSON et stockage dans des modèles adaptés
  • Développement de l'interface : création des écrans, listes et fiches détaillées des films
